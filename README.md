# APS-2025.1
Repositório usado para armazenar os artefatos pertinentes à análise e desenvolvimento de um projeto para instituição Universidade de Pernambuco, para a conclusão da disciplina de Análise e Projeto de Software

# Metodologia
A metodologia adotada para o andamento de todo o projeto será o RUP (Rational Unified Process)

# O que será o projeto?
O projeto a ser feito será o Sistema Integrado de TCCs, Estágio e Monitoria (SITEM).
